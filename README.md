# SafePath - Smart Tourist Safety Monitoring

A web application that combines geofencing, real-time location sharing, and incident alerts to keep travelers safe.

## Features

- **Real-time Location Tracking**: Share your location with trusted contacts
- **Smart Geofencing**: Create safe zones and get alerts when leaving them
- **SOS Emergency Button**: One-tap emergency alert to your safety circle
- **Safety Circle**: Add trusted contacts who can monitor your trips
- **Automated Alerts**: Get notified of geofence breaches and potential risks

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Mapping**: Leaflet.js with CartoDB Voyager (desaturated maps)
- **Backend**: Firebase (Authentication, Firestore)
- **Real-time**: Firebase Realtime updates
- **Styling**: Custom CSS with safety-focused color palette
